Interacción: alert, prompt, confirm

alert Muestra un mensaje.

prompt Muestra un mensaje pidiendo al usuario que introduzca un texto. Retorna el texto o, si se hace clic en CANCELAR o se presiona Esc, retorna null.

confirm Muestra un mensaje y espera a que el usuario pulse “OK” o “CANCELAR”. Retorna true si se presiona OK y false si se presiona CANCEL/Esc.

Crea una página web que pida un nombre y lo muestre.
